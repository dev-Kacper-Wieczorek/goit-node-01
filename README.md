# CLI Application for Contacts Management

## Features
- List all contacts
- Get contact by ID
- Add a new contact
- Remove a contact

## Commands
- `node index.js --action list`
- `node index.js --action get --id <id>`
- `node index.js --action add --name <name> --email <email> --phone <phone>`
- `node index.js --action remove --id <id>`
